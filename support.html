Camera or photo access is unavailable

Open the Settings app, find Rekeep, and allow Camera access. Photo selection uses Apple's system picker, so Rekeep only receives the photos you choose.

Reminders are not appearing

Confirm that notifications are enabled for Rekeep in system Settings. Then open Rekeep and check the relevant rediscovery or maintenance reminder settings.

iCloud data has not appeared yet

Check that both devices use the same Apple Account, iCloud Drive is enabled, and Rekeep has iCloud access. Keep Rekeep open briefly on a stable network while the first sync completes.

The widget is not refreshing

Open Rekeep once so it can prepare the latest widget content. If the widget remains unchanged, remove it from the Home Screen, restart the device, and add it again.

Before restoring a backup

Restoring a backup replaces the current Rekeep catalog. Export a fresh backup first if you may need the current version later, then select the file you want to restore.

Smart analysis or cutout needs improvement

Use a well-lit photo with the complete item visible and a clear boundary from the background. Smart suggestions are optional, and you can edit every result before saving.

Delete your data

You can delete individual records inside Rekeep. Local data is removed when the app and its data are deleted. If iCloud sync is enabled, manage the corresponding data through your iCloud storage settings.
